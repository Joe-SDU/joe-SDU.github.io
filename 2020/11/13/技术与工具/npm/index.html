npm介绍

npm，全称Node package manager，是一个方便快捷的包管理器。它的发展与node.js相辅相成。
什么是包管理器呢?其实理解起来很简单，就是帮助我们管理一大堆代码的仓库，有了它以后，我们需要啥代码就可以直接在里面拿。

npm的常用命令

  • npm install package

    此命令下载package

  • npm uninstall package

    此命令卸载package

  • npm ls

    此命令查看本地全部的package

  • npm search package

    此命令查询package的详细信息

  • npm -v

    此命令查看当前npm的版本
